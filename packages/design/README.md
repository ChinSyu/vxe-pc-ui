一个基于 [vue](https://www.npmjs.com/package/vue) 的低代码表单设计器  

* 设计理念
  * 面向现代浏览器，高效的简洁 API 设计
  * 支持 PC 端
  * 支持 H5 移动端
  * 支持小程序

* 计划
  * [] v3.0 基于 vue2.6，支持现代浏览器并保留兼容 IE11
  * [] v4.0 基于 vue3.x，只支持现代浏览器，不支持 IE
